My Projects

BMI & Calorie Intake Calculator

Role: Independent Developer

This project is a health-related web application that calculates a user daily calorie needs using the Harris-Benedict equation. Users can input personal information such as age, sex, weight, height, and activity level. The application dynamically processes the data and provides an estimated daily calorie intake based on the selected options.

Bug Smasher Game

Role: Independent Game Developer

Bug Smasher is an interactive browser-based game where users click on moving bugs to increase their score. The game includes real-time score tracking, adjustable game speed, and reset controls. JavaScript is used to manage animations, user interactions, and game logic.

Interactive Image Slideshow

Role: Independent Front-End Developer

This project is an interactive image slideshow that allows users to navigate through a photo gallery using previous and next controls. Additional features include play/pause functionality for automatic slideshow playback and the ability to view images in full-screen mode.
